TDA Luxury Mobil V5 - Müşteri İşlem Geçmişi

Önceki müşteri, randevu, kasa, paket, seans ve borç özelliklerinin tümünü içerir.

Kurulum:
1. App.tsx dosyasını mobil proje klasöründeki dosyanın üzerine yazın.
2. npm run typecheck
3. git add App.tsx
4. git commit -m "Müşteri işlem geçmişini ekle"
5. git push origin main
6. npx expo start -c

Yeni özellikler:
- Müşteri seans geçmişi
- Tahsilat geçmişi
- Ödeme yöntemi ve tutar
- Tarih ve saat
- Seans ve ödeme hareketlerini tek kronolojik listede gösterme
